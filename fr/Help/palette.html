Palette is an extension of main Canvas. We use it to prepare colors and patterns of Painting Tools. It shares Painting tools with main Canvas.

If you want to add new palette page or define custom colors, you can tap the in left bottom corner to switch to edition mode.

Multiple Palette Pages

Each palette page is a small size canvas, you can use it to blending colors, test painting tools and so on.

At present, you can allow to create 6 palette pages in one painting.

Custom Colors

You can tap the button to add custom color in edition mode, of course, you can tap button in right corner of it to remove it.

Special Actions

There are some special actions on palette:

  • Solvent, it's used like the colors in bowl, in real oil painting, it's like turpentine.

  • Clean current palette page.
