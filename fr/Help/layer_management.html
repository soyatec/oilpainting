Multiple layers support is a great feature of this application. It likes the real oil painting to allows you to drawing more layers in the same area.

Multiple Layers

With multiple layers support, the artwork will be stacked with layers, but only the top one layer can be drew. We called the top one to activated layer.

  • Add a new layer...

  • Merge selected layers.

  • Active the bottom layer of current activated layer.

  • Active the top layer of current activated layer.

Layer Control

Each layer can be hided, moved and cleaned.

  • Show/Hide current layer

  • Move current layer up, the index of current layer would be decrease 1.

  • Duplicate current layer.

  • Move current layer down, the index of current layer would be increase 1.

  • Clean current layer.

  • Remove current layer.

Focus Visibility

In order to distinguish each layer, we introduced the Focus Visibility value in Settings.

This value can change the transparency of the layer, it means that the transparency of activated layer is higher than all others.
