Overview

Gallery is used to display and manage all paintings, if iCloud storage is enabled, you can also upload all paintings to iCloud server or download items from iCloud server to your iPad.

Common

By default, you can create painting, sort paintings from the buttons on the top tool bar.
  • New

    Open the creation wizard to create a new painting, which contains the name, size and background settings of it.

  • Sorting

    By default, all paintings and their groups are sorted by most recently modified time, but you have 3 additional choice:

    1. Title: Sort all paintings with their title names.
    2. Date: Sort all paintings with the time of creation.
    3. Group: Sort all groups with the name of them, and then, you can move the paintings freely in edition mode.
  • Settings：

    Global settings of Gallery

  • In-App Purchase

    Open the store page to unlock painting tools, surfaces or some advanced features.

    Send questions, suggestions and feedback by email.

    Report bug once if you find.

Edition

In edition mode (tap 'Edit' button in navigation bar), you can manage all your groups and paintings, such as rename and delete.

You can also share and move paintings as you wanted:

  • Move

    A Painting can be moved freely if and only if they are sorted with 'Group'.

  • Upload to iCloud

    Upload all your selected paintings to iCloud server if enabled.

  • Download from iCloud

    Download all your selected paintings from iCloud server to local device.
