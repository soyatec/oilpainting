Gesture Description Settings
Single Finger Swipe left Undo Can be disabled from Settings
Single Finger Swipe right Redo Can be disabled from Settings
Single Finger Swipe up Clean current painting tool. Can be disabled from Settings
Single Finger Swipe down Enable Pattern picker to select existing pattern from canvas. Can be disabled from Settings
One finger in double Taps on screen Enable or Disable the Auto Reset option of the current tool. Can be disabled from Settings
Two Fingers in single Tap

Smart Zoom:

  • First tap to Fit to screen
  • Tap again to go back the previous zooming state
 Built-in
One finger in single Tap on Palette Force the focus on the Palette, it brings the Painting tool bar on top of Palette Built-in
One finger in single Tap on painting Canvas Show/hide the tool views such as tool bars, palette, light setting and tool controller Built-in
Two Fingers Pinch Zoom in/out Built-in
