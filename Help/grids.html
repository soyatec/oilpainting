This feature contains a list of assistance tools.

  • Grid

    Display grids on the top of the canvas.

  • Guide Lines

    Drawing some guide lines on the top of the canvas, but these lines are not the real part of the artwork.

  • Tracing

    Drawing the trace of a image.

  • Reference

    Open a image as reference while drawing.
